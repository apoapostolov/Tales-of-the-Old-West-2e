# 1870s: The Old West - Editorial Progress Tracker

This document serves as the master Table of Contents, editorial plan, and progress checklist for writing the massive 200+ page supplement _1870s: The Old West_ for Tales of the Old West 2e.

The source material is drawn from the `skills/western-writing/knowledge` database.
Vignettes will be integrated from `skills/western-writing/examples/scenes.md`.

## Editorial Rules

- **Formatting:** Break MD0013 (80-character line breaks). Use standard flowing paragraphs like the rest of `/corebook`. However, **DO NOT** convert tables into single paragraphs; preserve table formatting (though minimize the use of tables).
- **Prose Style:** The AI skill knowledge database must be converted from dry facts into interesting, captivating, and flavorful writing following the _Tales of the Old West 2e_ aesthetic and the `western-writing` skill guidelines (gritty, realistic, McCarthy/McMurtry/Portis tradition).
- **Factual Integrity:** Retain every single fact intact, even the uncomfortable ones. Just smoothen the presentation.
- **Integration:** Sprinkle example vignettes from `examples/scenes.md` throughout the book to set the mood where appropriate.
- **Goal:** A massive 200+ page setting supplement introducing players to the US in the 1870s.

## Work Procedure (For AI Assistant)

1. Mark this `todo.md` checklist using the `manage_todo_list` tool.
2. Read the corresponding source Markdown files from `skills/western-writing/knowledge`.
3. Read 1-2 examples from `skills/western-writing/examples/scenes.md` to insert as atmospheric sidebars/vignettes.
4. Rewrite the content in long flowing paragraphs in the desired flavorful prose. Keep all lists/tables and facts, but weave them better into narrative structures where possible.
5. Save the output to new `.md` files in `/1870s-the-old-west`. Name them sequentially (e.g., `00-introduction.md`, `01-peoples-and-conflict.md`, etc.).
6. Update `todo.md` progress continuously.
