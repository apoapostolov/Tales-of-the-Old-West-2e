# Farm and Holding Template

Use this template for farms, ranches, holdings, line camps, and other small sites that are too small for the full village statblock.

Do not use the old `Field / Value` location tables for these sites. Put the descriptive detail in prose paragraphs, then keep only an owner table and an amenity history table.

## [Holding Name]

[Holding name], in [county / district], is owned or worked by [owner / family / outfit]. It sits on [creek / bottomland / bench / road / range], and exists for [hay / stock / crops / timber / freight / supply / defense]. The nearest larger settlement is [town name]. In `1873`, it is [holding / improving / strained / exposed / isolated], and its survival depends on [water / feed / roads / labor / law / weather / market access].

Write the practical facts into the description paragraphs. Do not duplicate them in a second holding sheet.
