# Genre Signature Conflicts — Set-Piece Scripts

> **STATUS: WORKSHOP MODULE.** Reusable ceremonial conflict templates for duels, debates, trials, raids, dogfights, ritual contests, chases, infiltrations, courtroom scenes, and other genre-defining set pieces. Built from P3, P6, P8, AC4, and the conflict spine.

## 1. Origin

- **Source patterns:** P3, P6, P8, P15.
- **Operators:** Abstraction-Climb + Recalibration.
- **Target psychology:** the genre's iconic scene gets its own rhythm.
- **Problem solved:** a signature scene should not collapse into one generic roll.

## 2. Generic design space

A signature conflict is a 3-phase script. Each phase tests a different pressure. Margin carries forward as dice, position, or consequence. The final phase resolves the core stake.

### Universal script

1. **Read the field:** perception, social read, weather gauge, ritual omen, legal footing.
2. **Take position:** maneuver, argument, preparation, stance, sacrifice, approach.
3. **Commit:** strike, verdict, breach, broadside, confession, binding, escape.

Carry forward at most +2 dice or one advantage tag. Loser suffers the agreed stake or a typed fallout.

### Templates

| Conflict | Phase 1 | Phase 2 | Phase 3 | Fallout |
| --- | --- | --- | --- | --- |
| Duel | Face off | Draw/close | Strike | wound, shame, death |
| Debate | Read room | Frame issue | Deliver point | scandal, debt, concession |
| Trial | Establish standing | Present proof | Verdict | sentence, fine, exile |
| Raid | Scout | Breach | Seize/escape | alarm, injury, loot loss |
| Dogfight | Read vector | Maneuver | Fire/escape | vehicle crit, lost position |
| Ritual contest | Read omen | Channel | Bind/break | corruption, backlash |

### Resolution procedure

1. **Name the stake.** State exactly what victory and defeat mean before dice hit the table.
2. **Choose the script.** Pick or build a 2-5 phase template.
3. **Assign each phase a roll.** Each phase must test a different pressure: knowledge, position, nerve, resource, force, proof, sacrifice.
4. **Roll phases in order.** Opposed if an active opponent exists; threshold if the obstacle is procedural.
5. **Carry forward margin.** Winner gains +1 die per margin, max +2, or one advantage tag. A critical may also impose a complication on the loser.
6. **Fire the final stake.** The last phase decides the stake. Previous phases shape it; they do not replace it.
7. **Apply fallout.** Use existing harm/social/vehicle/power fallout or the generic set-piece fallout table.

| D6 | Fallout | Effect |
| --- | --- | --- |
| 1 | Costly victory | winner succeeds but spends resource/takes Stress |
| 2 | Public mark | Renown, Infamy, or Heat shifts by 1 |
| 3 | Damaged asset | gear, vehicle, witness, or Bond is harmed |
| 4 | Escalated enemy | loser gains revenge clock or faction support |
| 5 | Narrow escape | defeated side survives in worse position |
| 6 | Irreversible turn | death, exile, binding, verdict, wreck, or oath |

### Signature conflict design checks

- **Stake check:** if victory/defeat cannot be stated in one sentence, do not start.
- **Distinct phase check:** no two adjacent phases should roll the same skill for the same reason.
- **Carry cap check:** never carry more than +2 dice into a final phase without a cost.
- **Frequency check:** use for genre-defining scenes only; routine actions stay routine.

### Script construction procedure

To create a new signature conflict:

1. **Name the genre beat:** "ship broadside," "surgery under fire," "wizard duel," "boardroom coup."
2. **Name what ordinary rules miss:** timing, audience, preparation, proof, ritual order, crew roles.
3. **Pick 3 pressures:** one information pressure, one position/resource pressure, one commitment pressure.
4. **Assign failure cost per phase:** lost carry, complication, Heat, Stress, damaged asset, worse final stake.
5. **Choose fallout family:** harm, social, vehicle, project, reputation, territory, corruption.
6. **Write exit clause:** what happens if PCs concede after Phase 1 or 2?

### Phase cost menu

| Failed phase | Light cost | Heavy cost |
| --- | --- | --- |
| Read | opponent gets +1 carry | false read; final phase starts at −1 |

### Set-piece reward menu

Winning a signature conflict should grant a specific fiction-changing result.

| Reward | Use when |
| --- | --- |
| **Decisive position** | next normal conflict starts with range/cover/objective advantage |
| **Public verdict** | audience/faction accepts outcome until new evidence |
| **Unlocked route** | travel/domain obstacle removed |
| **Bound threat** | monster/power changes phase or becomes containable |
| **Claimed asset** | ship, territory, witness, relic, office secured |
| **Changed relationship** | Bond, Debt, Influence, Reputation shifts permanently |

### Worked mini-build

**Exorcism:** Stake: save the host without letting the spirit escape. Phase 1 NAME SPIRIT (Lore/Insight) reveals weakness; failure gives spirit +1. Phase 2 ANCHOR HOST (Empathy/Faith) prevents host harm; failure marks Bond Strain or Stress. Phase 3 COMMAND OUT (Power/Social) resolves. Optional Phase 4 SEAL DOOR prevents recurrence; failure starts a Corruption clock. Fallout uses Stress, Corruption, and Bond tables rather than weapon harm.

## 3. Pressure loop

Stake -> phased commitments -> carry-forward advantage -> decisive final roll -> fallout.

## 4. Choices

| Choice | Light | Heavy | Psychology |
| --- | --- | --- | --- |
| Phases | 2 | 3-5 | quick drama vs ceremony |
| Carry-forward | +1 cap | +2/tag cap | grounded vs decisive |
| Fallout | existing harm | custom table | generic vs genre-rich |

## 5. Integration points

Hooks into combat, social combat, chases, vehicles, magic, territory, and reputation.

## 6. Handshake

- **Prerequisites:** opposed rolls and stakes.
- **Inputs:** signature scene, three skills/pressures, fallout.
- **Outputs:** phase script, carry-forward rule, final stake.
- **Touched systems:** conflict, harm/social/vehicle/power as selected.
- **Replaces or stacks:** replaces ordinary conflict only for the named scene type.
- **Incompatibilities:** do not use for routine actions.

## 7. Failure modes

If used too often, ceremony becomes drag. If carry-forward is uncapped, Phase 1 decides everything. If the final stake is unclear, players cannot consent to risk.

## 8. Check notes

Use `24 §1`, `§4`, and `§7`. Carry-forward dice are strong; cap them and keep phases distinct.

## 9. Publication handoff

**Player rule:** When the table enters a signature conflict, agree the stake, play the phases in order, and let each phase shape the last.

## 10. Worked example — Courtroom trial

Phase 1: Establish standing. Phase 2: Present proof. Phase 3: Verdict. A failed final roll creates sentence, fine, exile, or debt; a narrow success still creates Heat with the losing faction.
